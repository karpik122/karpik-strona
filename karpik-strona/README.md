# Karpik Strona - Portfolio

Welcome to the Karpik Strona project! This is a personal portfolio website showcasing graphic and web design projects, along with a contact section and a detailed CV.

## Project Structure

The project is organized as follows:

```
karpik-strona
├── src
│   ├── index.html          # Main homepage of the portfolio
│   ├── contact.html        # Contact information and form
│   ├── gallery-art.html    # Gallery showcasing art projects
│   ├── cv.html             # Curriculum Vitae of the portfolio owner
│   ├── css
│   │   └── style.css       # Styles for the entire project
│   └── js
│       └── main.js         # JavaScript functionality for the project
└── README.md               # Project documentation
```

## Features

- **Homepage**: An introduction to the portfolio owner, showcasing projects and future development plans.
- **Contact Page**: A dedicated section for users to reach out via a contact form or view contact information.
- **Gallery Art Page**: A visual display of art projects with images and descriptions.
- **CV Page**: A detailed overview of the portfolio owner's education, experience, and skills.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `src` directory.
3. Open `index.html` in your web browser to view the homepage.
4. Navigate to other pages using the links provided in the navigation bar.
